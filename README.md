# Reporting Hub - White Label BI Platform

A modern Next.js landing page for Reporting Hub, a Microsoft-endorsed white label BI platform that transforms Power BI dashboards into revenue-driving SaaS solutions.

## Features

- ✅ Share with unlimited users, no license hassle
- ✅ No-code, lightning-fast deployment
- ✅ Fully brand your analytics portals

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first CSS framework
- **React 18** - Modern React features

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### Build for Production

```bash
npm run build
npm start
```
